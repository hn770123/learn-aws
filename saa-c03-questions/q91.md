# SAA-C03 問題: AWS CodeArtifact によるアーティファクトリポジトリの管理

## 問題
ソフトウェア開発において使用されるライブラリやパッケージ（npm, PyPI, Maven 等）を安全に保存、公開、および共有し、パブリックリポジトリからの依存関係を一元的に管理するためのサービスはどれですか？

## 選択肢
A. Amazon ECR
B. AWS CodeArtifact
C. AWS CodeCommit
D. AWS CodeBuild

## 正解
B

## 解説
AWS CodeArtifact は、あらゆる規模の組織が、ソフトウェア開発プロセスで使用されるソフトウェアパッケージを安全に保存、公開、および共有できるようにするフルマネージド型のアーティファクトリポジトリサービスです。

*   **Cについて**: ソースコード（Git）用です。
*   **Dについて**: ビルド実行用です。
*   **Aについて**: コンテナイメージ用です。
