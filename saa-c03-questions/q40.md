# SAA-C03 問題: AWS App Mesh によるマイクロサービス間の通信管理

## 問題
多数のマイクロサービス間の通信を標準化し、一貫した可視性、トラフィック制御、およびセキュリティを提供するためのサービスメッシュ機能を提供するサービスはどれですか？

## 選択肢
A. AWS App Mesh
B. Elastic Load Balancing (ELB)
C. Amazon API Gateway
D. AWS Cloud Map

## 正解
A

## 解説
AWS App Mesh は、サービスメッシュを提供し、マイクロサービス間の通信を容易に管理できるようにします。サイドカープロキシ (Envoy) を使用して、アプリケーションコードを変更せずに、通信の可視化、再試行ロジック、トラフィックの分散、暗号化を実現します。

*   **Bについて**: ELB は外部からのトラフィック分散が主目的です。
*   **Cについて**: API Gateway は外部に公開する API の管理です。
*   **Dについて**: Cloud Map はリソースの検出（サービスディスカバリ）サービスです。
