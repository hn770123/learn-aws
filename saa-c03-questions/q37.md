# SAA-C03 問題: Amazon Macie による S3 上の機密データの自動検出

## 問題
Amazon S3 に保存されている大量のデータの中から、個人情報 (PII) や機密データを機械学習を用いて自動的に検出し、保護するためのサービスはどれですか？

## 選択肢
A. AWS KMS
B. Amazon Macie
C. Amazon GuardDuty
D. AWS CloudTrail

## 正解
B

## 解説
Amazon Macie は、フルマネージドなデータセキュリティおよびデータプライバシーサービスで、機械学習とパターンマッチングを使用して Amazon S3 内の機密データを自動的に検出、分類、および保護します。

*   **Cについて**: GuardDuty はアカウントやネットワークレベルの脅威検出です。
*   **Aについて**: KMS は暗号化キーの管理サービスです。
*   **Dについて**: CloudTrail は API ログの記録です。
