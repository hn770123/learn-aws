# SAA-C03 問題: Amazon Cognito による Web/モバイルアプリのユーザー認証と認可

## 問題
Web やモバイルアプリケーションに、サインアップ、サインイン、およびアクセスコントロールの機能を簡単に追加したいと考えています。ソーシャルログイン（Google, Facebook等）や、SAML 2.0 による ID 連携をサポートしているサービスはどれですか？

## 選択肢
A. AWS IAM
B. Amazon Cognito
C. AWS Organizations
D. AWS Directory Service

## 正解
B

## 解説
Amazon Cognito を使用すると、Web アプリやモバイルアプリに迅速かつ簡単にユーザーのサインアップ/サインインおよびアクセスコントロール機能を追加できます。ユーザープール（ID 保存）と ID プール（AWS リソースへのアクセス許可）の機能を提供します。

*   **Aについて**: IAM は主に AWS リソースを管理する従業員等の権限管理に使用されます。
*   **Cについて**: Organizations は複数アカウントの管理用です。
*   **Dについて**: Directory Service は Active Directory などを AWS 上で展開・連携するためのものです。
