# SAA-C03 問題: AWS WAF を使用した Web アプリケーションへの一般的な攻撃の防御

## 問題
Application Load Balancer (ALB) の背後で稼働している Web アプリケーションを、SQL インジェクションやクロスサイトスクリプティング (XSS) などの一般的な Web 攻撃から保護したいと考えています。どのサービスを導入すべきですか？

## 選択肢
A. AWS Shield Advanced
B. AWS WAF
C. Amazon Inspector
D. Amazon GuardDuty

## 正解
B

## 解説
AWS WAF (Web Application Firewall) は、ALB、Amazon CloudFront、Amazon API Gateway などにデプロイでき、SQL インジェクションや XSS などの一般的なウェブ攻撃をブロックするためのルールを設定できます。

*   **Aについて**: AWS Shield は主に DDoS 攻撃からの保護を提供します。
*   **Cについて**: Amazon Inspector は EC2 インスタンスやコンテナの脆弱性を診断するサービスです。
*   **Dについて**: Amazon GuardDuty は AWS アカウントやリソースに対する脅威を継続的にモニタリングするサービスですが、インラインでの攻撃ブロック（ファイアウォール機能）は提供しません。
