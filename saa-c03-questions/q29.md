# SAA-C03 問題: AWS Transfer Family による S3 へのセキュアなファイル転送

## 問題
既存のビジネスプロセスで SFTP プロトコルを使用してファイルを交換しています。インフラの管理をせずに、SFTP 経由で直接 Amazon S3 にファイルをアップロード・ダウンロードできるようにするには、どのサービスを使用すべきですか？

## 選択肢
A. AWS Storage Gateway
B. AWS Transfer Family
C. AWS DataSync
D. Amazon S3 マルチパートアップロード

## 正解
B

## 解説
AWS Transfer Family は、SFTP、FTPS、および FTP を使用して Amazon S3 または Amazon EFS との間でファイルを直接転送できるようにするフルマネージドサービスです。既存のプロトコルを維持しつつ、サーバー管理の負担をなくすことができます。

*   **Aについて**: Storage Gateway も NFS/SMB を提供しますが、SFTP サーバーをマネージドで提供するわけではありません。
*   **Cについて**: DataSync はオンラインデータ転送の高速化ツールであり、定常的な SFTP インターフェースを提供するものではありません。
*   **Dについて**: マルチパートアップロードは S3 API の機能であり、SFTP プロトコルとは異なります。
