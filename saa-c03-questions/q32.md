# SAA-C03 問題: AWS AppSync を使用したマネージドな GraphQL API の構築

## 問題
モバイルおよび Web アプリケーション向けに、複数のデータソース（DynamoDB、Lambda、HTTP API）からのデータを統合し、リアルタイムの更新機能を提供する GraphQL API を構築する必要があります。どのサービスが最適ですか？

## 選択肢
A. Amazon API Gateway
B. AWS AppSync
C. AWS Amplify
D. Amazon QuickSight

## 正解
B

## 解説
AWS AppSync は、GraphQL を使用してアプリケーションが正確に必要なデータだけに簡単にアクセスできるようにするフルマネージドサービスです。複数のデータソースからのデータ集約を容易にし、WebSocket を使用したリアルタイムのデータ同期をネイティブにサポートしています。

*   **Aについて**: API Gateway は REST API の構築には適していますが、GraphQL 特有の機能や複雑なデータ集約を AppSync ほど容易には提供しません。
*   **Cについて**: Amplify はフレームワークおよびツールセットであり、バックエンドとして AppSync を利用しますが、API サービス自体は AppSync です。
*   **Dについて**: QuickSight はビジネスインテリジェンス（可視化）ツールです。
