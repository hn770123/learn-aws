# SAA-C03 問題: AWS Supply Chain によるサプライチェーンの可視化

## 問題
複数の既存システムからデータを統合し、在庫の状況や配送の遅延リスクを機械学習で可視化・予測することで、サプライチェーンの回復力を高めるためのクラウド型アプリケーションはどれですか？

## 選択肢
A. Amazon QuickSight
B. Amazon Forecast
C. AWS Supply Chain
D. AWS AppFlow

## 正解
C

## 解説
AWS Supply Chain は、既存のサプライチェーンシステムからデータを統合、可視化、分析し、在庫切れのリスクやリードタイムの変動を機械学習ベースの予測機能で通知するアプリケーションです。

*   **Bについて**: 時系列データ予測全般のためのサービスです。
