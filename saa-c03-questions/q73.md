# SAA-C03 問題: AWS Mainframe Modernization によるメインフレーム資産の移行

## 問題
既存のメインフレームワークロードを AWS のクラウドネイティブなランタイム環境に移行、モダナイズ、および実行するために設計されたサービスはどれですか？

## 選択肢
A. AWS Database Migration Service (DMS)
B. AWS Mainframe Modernization
C. AWS Application Migration Service (MGN)
D. AWS DataSync

## 正解
B

## 解説
AWS Mainframe Modernization は、メインフレームアプリケーションを AWS に移行、モダナイズ、および実行するためのツールとランタイム環境を提供するマネージドサービスです。リプラットフォームやリファクタリング（Java への変換など）を支援します。

*   **Cについて**: サーバー（x86 等）の単純移行（リホスト）用です。
*   **Dについて**: データ転送用です。
*   **Aについて**: データベースの移行用です。
