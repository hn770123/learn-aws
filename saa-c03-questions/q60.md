# SAA-C03 問題: Amazon Honeycode によるノーコードアプリ開発

## 問題
プログラミングコードを書くことなく、スプレッドシートのような操作感でカスタムの Web およびモバイルアプリケーションを作成し、チームの業務プロセスを管理したいと考えています。どのサービスを使用すべきですか？（※注：現在は新規受付終了またはサービス終了の方向にある場合がありますが、試験範囲に含まれる可能性があります）

## 選択肢
A. AWS Amplify
B. AWS AppSync
C. Amazon QuickSight
D. Amazon Honeycode

## 正解
D

## 解説
Amazon Honeycode は、プログラミングを学ばなくてもカスタムアプリを構築できるノーコードプラットフォームです。スプレッドシートモデルに基づいて、データの管理や通知機能を備えたアプリを作成できます。

*   **Bについて**: AppSync は GraphQL API 開発用です。
*   **Aについて**: Amplify は開発者向けのフレームワークです。
*   **Cについて**: QuickSight はデータ可視化ツールです。
