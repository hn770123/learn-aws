# SAA-C03 問題: AWS Wickr によるセキュアなエンタープライズ通信

## 問題
エンドツーエンドの暗号化を備え、機密性の高い通信（メッセージング、通話、ファイル共有）を組織内で安全に行うためのエンタープライズ向けサービスはどれですか？

## 選択肢
A. Amazon Chime
B. Amazon Connect
C. Amazon SES
D. AWS Wickr

## 正解
D

## 解説
AWS Wickr は、256 ビットのエンドツーエンド暗号化を使用して、エンタープライズや政府機関が安全に共同作業を行えるように設計された、安全性重視のコミュニケーションサービスです。

*   **Aについて**: ビデオ会議・コミュニケーションツールですが、Wickr ほど極端な機密保持（エンドツーエンド暗号化の徹底）に特化した設計ではありません。
*   **Cについて**: メールサービスです。
*   **Bについて**: コンタクトセンターサービスです。
