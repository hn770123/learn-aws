# SAA-C03 問題: AWS Elastic Disaster Recovery (DRS) による迅速な災害復旧

## 問題
オンプレミスまたは AWS 上の物理サーバー、仮想サーバーを、低コストのストレージを使用して AWS へ継続的にレプリケートし、障害発生時に数分以内に迅速に復旧できるようにするための災害復旧（DR）サービスはどれですか？

## 選択肢
A. AWS Backup
B. AWS Elastic Disaster Recovery (DRS)
C. AWS CloudFormation
D. AWS Site-to-Site VPN

## 正解
B

## 解説
AWS Elastic Disaster Recovery (DRS) は、アプリケーションのダウンタイムとデータ損失を最小限に抑え、オンプレミスおよびクラウドベースのアプリケーションを迅速かつ確実なリカバリで稼働させることができます。ステージングエリアにデータをレプリケートし、必要時のみインスタンスを起動するためコスト効率に優れています。

*   **Aについて**: バックアップ（スナップショット）からの復元は時間がかかることがあり、DRS のような継続的なレプリケーションと迅速なフェイルオーバーとは異なります。
*   **Cについて**: インフラの構築用です。
*   **Dについて**: 接続用です。
