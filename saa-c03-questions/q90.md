# SAA-C03 問題: Amazon Bedrock による生成 AI アプリケーションの構築

## 問題
AI21 Labs, Anthropic, Cohere, Meta, Mistral AI, Stability AI, および Amazon の基盤モデル (FM) を API 経由で利用し、生成 AI アプリケーションを迅速に構築・スケールするためのフルマネージドサービスはどれですか？

## 選択肢
A. Amazon Rekognition
B. Amazon SageMaker
C. Amazon Bedrock
D. Amazon Kendra

## 正解
C

## 解説
Amazon Bedrock は、大手 AI 企業の基盤モデルを API を通じて利用できるようにするサービスです。サーバーレスであるため、インフラ管理なしで、プロンプトエンジニアリングや検索拡張生成 (RAG) などを活用した生成 AI アプリケーションを構築できます。

*   **Bについて**: モデルを一から訓練したり微調整してデプロイしたりする用途です。
*   **Cについて**: 学習済みの強力な基盤モデルを即座に利用する用途に最適化されています。
