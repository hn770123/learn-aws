# SAA-C03 問題: AWS CloudTrail によるユーザーアクティビティと API コールのログ記録

## 問題
AWS アカウント全体のガバナンス、コンプライアンス、およびリスク監査のために、誰が、いつ、どのリソースに対して、どの API を呼び出したかを記録・保存するためのサービスはどれですか？

## 選択肢
A. Amazon CloudWatch
B. AWS Config
C. AWS CloudTrail
D. AWS Trusted Advisor

## 正解
C

## 解説
AWS CloudTrail は、AWS アカウントのガバナンス、コンプライアンス、操作監査、リスク監査を可能にするサービスです。ユーザー、ロール、または AWS サービスによって実行されたアクション（API コール）をイベントとしてログに記録します。

*   **Aについて**: CloudWatch はパフォーマンスメトリクスやログメッセージの監視用です。
*   **Bについて**: Config はリソースの設定状態（Inventory）の記録と評価用です。
*   **Dについて**: Trusted Advisor はベストプラクティスのアドバイスを提供します。
